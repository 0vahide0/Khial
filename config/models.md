# Canonical model names and options for khial

# Image models
- nanobanana (default image model)

# Thinking / reasoning / decision models
- gemini-3 (high-quality reasoning & decisions)
- gemini-3-flash (faster variant of gemini-3)
- gemini-3-lite (low-cost, very fast simple tasks)

# TTS models
- gemini-tts-pro-2.5 (high-quality expressive TTS)
- gemini-tts-flash (lower-latency TTS variant)

# Notes
- The Planner will present these options and let the user choose per-step which model to call.
